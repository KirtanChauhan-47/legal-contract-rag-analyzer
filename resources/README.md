# Sample documents

Persistent test fixtures for manual/live verification during development —
not regenerated or deleted between sessions. Drop additional sample
contracts (PDF/DOCX/TXT) here as needed; committed to git so fixtures stay
available across sessions instead of being recreated from scratch each time.

- `sample_nda.txt` — a realistic multi-clause NDA, used to verify the
  accept path (contract gate, chunking, embedding, Q&A, clause detection).
- `not_a_contract.txt` — a non-legal text, used to verify the contract
  gate's rejection path.
